# LiteLLM Control Plane

AI Resource Controller & control plane built on LiteLLM. Orchestrates multi-region VPS nodes, Tailscale exit nodes, and multi-provider/model/account pools with health checks, cooldowns, cost-aware routing, and quotas—without replacing LiteLLM’s gateway, only governing it.

## Directory Structure

Refer to the source files in `src/litellm_control_plane` and documentation in `docs/`.
